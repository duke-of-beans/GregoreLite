Sprint S9-10 (Cost Breakdown by Project) adds a per-project cost view. CostBreakdown.tsx is a modal panel triggered by clicking the daily cost in StatusBar, with three tabs (Today, This Week, All Time). Each tab shows a table of project_id, session count, input tokens, output tokens, and cost USD — sorted by cost descending with a total row at bottom. The API endpoint GET /api/costs/breakdown aggregates session_costs grouped by project_id with time-range filtering via strftime conditions. StatusBar.tsx was modified to open the CostBreakdown modal on cost click instead of switching tabs. Empty state renders cleanly when no data exists. Quality gates: tsc clean, 890/890 tests pass.
