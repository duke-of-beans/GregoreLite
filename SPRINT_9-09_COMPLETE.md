Sprint S9-09 (EoS History Sparkline) adds trend visualization to the Context Panel's quality section. EoSSparkLine.tsx renders an 80×24px SVG line chart of the last 30 health scores from eos_reports, colored green (≥80), amber (60-79), or red (<60) based on the current score. Current score and delta vs previous scan are shown inline ("85 (+3)"). When fewer than 2 data points exist, it falls back to plain score text. Clicking the sparkline opens EoSHistoryPanel.tsx, a 420px fixed drawer showing the last 50 scans in a table with date, score, delta, critical count, warning count, and scan mode. Clicking any row expands to show the full issues_json. The API endpoint GET /api/eos/history returns eos_reports rows for a given project_id. ContextPanel.tsx was modified to mount EoSSparkLine in place of the old score text, and to mount EoSHistoryPanel when opened. Quality gates: tsc clean, 890/890 tests pass.
